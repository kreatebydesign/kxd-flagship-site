# KXD Brand Direction

## Reference

**Primary visual reference:** [kreatebydesign.com](https://kreatebydesign.com)

This rebuild is an evolution of the current live site — sharper, richer, cleaner, more scalable. **Not a rebrand.**

## Brand DNA

KXD is dark, bold, sharp, luxury, confident.

| Attribute | Expression |
|-----------|------------|
| Atmosphere | Black backgrounds, warm cream/white type, gold accents |
| Headlines | Bold uppercase editorial — stacked display type |
| Energy | Precision. Clarity. Presence. |
| Tone | Direct, premium, studio-grade — never SaaS, never generic |
| CTAs | Gold gradient or ghost white — uppercase, wide tracking |

## Signature Copy (preserve)

- **LUXURY WEBDESIGN** — stacked hero display
- **Precision. Clarity. Presence.**
- **This is digital luxury by KXD.**
- **Built Beyond Standards.**
- **Proof Over Promises.**
- **Let's Build What Others Can't.**

## What We Upgrade

- Cleaner information architecture
- Payload-driven portfolio and case studies
- Stronger SEO and conversion paths
- Scalable design tokens and component foundation
- More polished motion and spacing discipline

## What We Do Not Change

- The recognizable KXD visual identity
- Black/gold/cream contrast language
- Uppercase editorial headline system
- Dark luxury atmosphere
- Sharp studio confidence

## What to Avoid

- Generic Apple/Stripe minimalism that erases KXD character
- Navy-blue SaaS backgrounds (live site is **black**, not navy)
- Soft sentence-case heroes
- Geist/Inter system-font feel
- Blue primary CTAs (gold is the live brand action color)
- Excessive white space and corporate spacing
- Rounded friendly card aesthetics
- Copy that sounds like a different company

## Color Authority

Sourced from live site CSS:

| Token | Value | Role |
|-------|-------|------|
| Background | `#0a0a0a` | Page base |
| Background deep | `#070707` | Gradient depth |
| Foreground | `#ffffff` | Primary type |
| Cream | `#faf8f5` | Warm secondary type |
| Gold | `#c9a962` | Accent, labels, rules |
| Gold light | `#d4ba7a` | Hover, highlights |
| Muted | `#888888` | Supporting copy |
| Border | `#2a2a2a` | Cards, dividers |
| Card | `#111111` | Case study surfaces |

Blue (`#285bff`) and crimson (`#d94141`) remain in the extended palette for rare brand moments — not as default UI colors.

## Typography Authority

| Role | Font | Live site |
|------|------|-----------|
| Display | Cormorant Garamond | Hero, section headlines |
| UI / Body | Outfit | Navigation, body, labels, buttons |

Headlines: uppercase, wide tracking, light-to-regular weight on display.

## Strategic Positioning (unchanged)

1. **Luxury Websites** — primary entry point
2. Ecommerce, Growth Infrastructure, Operational Platforms, Enterprise Systems — premium expansion

KXD OS remains separate. Do not merge operational platform UI into the marketing site.

## Design Standard

> The current KXD website evolved into a million-dollar studio site.

Not new brand. Not generic premium. **KXD — sharper, richer, cleaner, more dangerous.**
