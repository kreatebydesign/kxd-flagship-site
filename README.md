# Kreate by Design — Flagship Rebuild

Premium creative and digital infrastructure studio website. Built on Next.js 16 + Payload CMS 3.

## Quick Start

```bash
cp .env.example .env.local
npm install
npm run dev
```

- **Site**: http://localhost:3000
- **Admin**: http://localhost:3000/admin

SQLite is used automatically in development when `DATABASE_URI` is not set.
